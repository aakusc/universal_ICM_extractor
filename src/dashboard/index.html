How It Works

The Universal ICM Connector extracts compensation rule concepts from any ICM vendor system and normalizes them into a single schema that downstream tools can consume.

  1. 1
    Connect Authenticate against the target ICM system (Varicent, Xactly, SAP, CaptivateIQ, Salesforce) via OAuth, API key, or SAML.
  2. 2
    Extract Rules Pull raw compensation plan rules from the vendor API — rate tables, accelerators, qualifiers, splits, territories, and more.
  3. 3
    Interpret Concepts AI analyzes each raw rule and identifies the business intent (e.g. “accelerator above 100% quota”) rather than copying vendor-specific objects. Each rule is classified into one of 12 concept types with a confidence score.
  4. 4
    Normalize Output to a vendor-agnostic JSON schema (validated with Zod) consumable by the Commission Calculator, Pay Curves, Modeling & Simulation, and SGM/SPARCC.
